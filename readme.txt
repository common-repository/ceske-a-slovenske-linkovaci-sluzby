Tested up to: 2.8.5
Stable tag: 0.3

České a slovenské linkovací služby jako doplněk pluginu Sociable.

== Description ==

České a slovenské linkovací služby (Linkuj.cz, Jagg.cz, MediaBlog.cz, TOPodkazy.cz, Bookmarky.cz, Mojelinky.sk, vybrali.sme.sk, Pozrisi.sk, Park.sk, TextTip.eu, Top Články.cz a Blogus.cz) jako doplněk pluginu [Sociable](http://yoast.com/wordpress/sociable/).
Tento plugin tedy vyžaduje nainstalovaný plugin Sociable (verze 3.4 a vyšší).

Pokud chcete přidat další službu, neváhejte o ni požádat [zde](http://www.separatista.net/wordpress/pluginy/ceske-a-slovenske-linkovaci-sluzby-jako-samostatny-plugin-ve-wordpressu).

== Installation ==

**Instalace**

1. Nejdříve najděte a naistalujte plugin [Sociable](http://wordpress.org/extend/plugins/sociable/).
2. Poté nainstalujte plugin České a slovenské linkovací služby.
3. V menu Nastavení - Sociable uvidíte v nabídce i české a slovenské linkovací služby, ze kterých si můžete libovolně vybírat.
